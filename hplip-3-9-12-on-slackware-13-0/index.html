hplip 3.9.12 on Slackware 13.0

I've got a new HP printer, HP F2480, which it wasn't  supported by the older hplip 3.9.4b version avaible for Slackware 13.0

So I've build the latest 3.9.12 version modifying the SlackBuild avaible here: ftp://ftp.slackware.com/pub/slackware/slackware_source/ap/hplip/ .

For version 3.9.12  that patch is no longer needed so I've removed it.

To build the new version you'll need to download the [[hplip]] source avaible here: http://prdownloads.sourceforge.net/hplip/hplip-3.9.12.tar.gz

and also the modified SlackBuild avaible here: http://oprod.net/slackware/hplip/slackbuild/
